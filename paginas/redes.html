Redes Informáticas

Las redes informáticas son sistemas de comunicación que permiten la interconexión de dispositivos para compartir recursos e información.

1. Tipos de Redes

1.1 Según su alcance

  • Red de Área Personal (PAN): Conexión entre dispositivos cercanos (Bluetooth).
  • Red de Área Local (LAN): Conecta equipos en un área reducida (oficinas, hogares).
  • Red de Área Metropolitana (MAN): Cubre ciudades o campus universitarios.
  • Red de Área Amplia (WAN): Conecta redes a nivel global (Internet).

1.2 Según su estructura

  • Red Punto a Punto: Conexión directa entre dos dispositivos.
  • Red Cliente-Servidor: Un servidor proporciona servicios a múltiples clientes.
  • Red Peer-to-Peer (P2P): Todos los dispositivos actúan como clientes y servidores.

2. Protocolos de Comunicación

  • TCP/IP: Base de la comunicación en Internet.
  • HTTP/HTTPS: Protocolo para la navegación web.
  • FTP: Transferencia de archivos.
  • DNS: Traducción de nombres de dominio a direcciones IP.
  • SMTP/POP3/IMAP: Protocolos para correo electrónico.

3. Seguridad en Redes

3.1 Amenazas Comunes

  • Intercepción de datos: Espionaje y robo de información.
  • Ataques de denegación de servicio (DDoS): Saturación de servidores.
  • Malware y phishing: Software malicioso y fraudes.

3.2 Medidas de Seguridad

  • Firewall: Filtrado de tráfico no autorizado.
  • VPN: Comunicación segura mediante cifrado.
  • Autenticación robusta: Uso de MFA y contraseñas seguras.
  • Monitoreo de red: Detección temprana de intrusiones.

4. Tendencias en Redes Informáticas

  • Redes 5G: Mayor velocidad y conectividad.
  • SDN (Redes Definidas por Software): Control centralizado de redes.
  • Computación en el Borde (Edge Computing): Procesamiento de datos cercano a la fuente.
  • IoT (Internet de las Cosas): Interconexión de dispositivos inteligentes.
