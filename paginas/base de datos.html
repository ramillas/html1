Bases de Datos

Las bases de datos son sistemas organizados para almacenar, gestionar y recuperar información de manera eficiente.

1. Tipos de Bases de Datos

1.1 Según su Modelo

  • Relacionales: Organizan datos en tablas con claves primarias y relaciones (Ej. MySQL, PostgreSQL).
  • NoSQL: Almacenan datos de forma flexible en documentos, grafos o clave-valor (Ej. MongoDB, Redis).
  • Jerárquicas: Estructura de árbol con relaciones padre-hijo.
  • Red: Similar a la jerárquica pero con múltiples conexiones.

1.2 Según su Ubicación

  • Locales: Almacenadas en un único servidor.
  • Distribuidas: Repartidas en varios servidores conectados en red.
  • En la Nube: Servicios como Amazon RDS, Google Cloud SQL.

2. Lenguajes de Bases de Datos

  • SQL: Lenguaje estándar para bases de datos relacionales.
  • PL/SQL: Extensión de SQL usada en Oracle.
  • Mongo Query Language (MQL): Para bases NoSQL como MongoDB.

3. Seguridad en Bases de Datos

3.1 Amenazas Comunes

  • Inyección SQL: Inserción de código malicioso.
  • Accesos no autorizados: Explotación de vulnerabilidades.
  • Pérdida de datos: Fallos en hardware o ataques cibernéticos.

3.2 Medidas de Seguridad

  • Control de Accesos: Roles y permisos.
  • Cifrado de Datos: Protección contra accesos no autorizados.
  • Copias de Seguridad: Prevención ante pérdida de información.
  • Monitoreo y Auditoría: Detección de accesos sospechosos.

4. Tendencias en Bases de Datos

  • Big Data: Procesamiento de grandes volúmenes de datos.
  • Bases de Datos en Memoria: Mayor velocidad en consultas.
  • Blockchain: Seguridad y trazabilidad de la información.
  • Inteligencia Artificial: Optimización y automatización de consultas.
